# sestodimpianto
QGIS Processing script per progettare sesti d'impianto arborei. Genera punti pianta (rettangolo/quinconce) con capezzagna, zone di esclusione, impollinatori, tutori distribuiti uniformemente, pali di testata e linee fila. Stima fabbisogno materiale completa. Export multi-layer KML e GeoPackage per navigazione RTK in campo con SW Maps.
